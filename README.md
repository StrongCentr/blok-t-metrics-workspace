# Рабочее место лидера Блок Т — v2.0

Статический прототип: главная dashboard лидера, пирамида метрик L1–L4, продуктовая иерархия, описания листовых продуктов. Mock-данные.

**Сайт:** https://strongcentr.github.io/blok-t-metrics-workspace/

## Локально

Откройте `Курсор.html` или `index.html` в браузере.

## Обновление GitHub Pages

```powershell
.\deploy-github-pages.ps1
```

Скрипт копирует актуальный HTML в `index.html`, коммитит и пушит в `main`.
